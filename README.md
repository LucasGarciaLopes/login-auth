# Sistema de Login e Cadastro Full Stack

Aplicação Full Stack com autenticação de usuários desenvolvida com HTML, CSS, JavaScript, Node.js e Express.

## 🚀 Funcionalidades
- Cadastro de usuário
- Login
- Proteção de rota (dashboard)
- Persistência em arquivo JSON
- Comunicação via Fetch API
- Controle de sessão com localStorage

## 🛠️ Tecnologias
- Node.js
- Express
- HTML
- CSS
- JavaScript

## ▶️ Como rodar

npm install
node server.js

Acesse:
http://localhost:3000/login.html
